L'application ci jointe a été crée dans le cadre d'un projet de GLA de L3 Informatique/MIAGE. 
Celle ci a été développée conjointement par Samia BEN OUARAB, Tom TRING et Sophiane LAROUB.

Cette application permet de créer des profils utilisateurs, puis de génerer une soirée correspondant aux adresses des profils inserés.
Cette application permet les actions suivantes :

- Creer un profil
- Modifier un profil
- Supprimer un profil
- Génerer/Relancer une soirée
- Envoyer un récapitulatif de la soirée par mail aux participants.

Cette application a été développée a l'aide d'Android Studio dont certaines parties en drag and drop.
Certaines fonctions de cette application nécéssitent un niveau d'API minimum de 22 (Lollipop).

Le dossier src contient l'application en format compressé (ZIP).

Pour executer l'application (sous Android Studio), file -> open -> appliBar (le répertoire une fois décompressé).
